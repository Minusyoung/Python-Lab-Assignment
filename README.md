# Python-Lab-Assignment
基于 6D-Delta LSTM 的预测性资产流送调度框架
# 实验环境：
 - 系统环境：windows 11
 - 语言：python3.9
 - 开发环境：pycharm 2025.3
 - 所用核心库：numpy pandas mcrcon pydirectinput matplotlib seaborn
 - torch版本：2.11.0
 - 辅助工具：Gemini pro
 - 游戏环境：Minecraft java 1.20.1 + 地平线模组
 - 数据集：手搓的
----------------------------------------
## 作者的话
  这个项目的灵感来源于游玩我的世界整合包期间。当时在游戏里跑图十分痛苦，经常速度太快要停下来等区块加载 ~（服主禁了op权限）~ 。当时我正好在学习lstm模型，于是突发奇想，思考能不能打破原来的地图加载方式，来提高跑图速率，这就是这个项目的雏形。一开始做的很简陋，只能预测长距离轨迹。后来借助Gemini  ~（真好用）~ ，将预测变量增加到了6个。即使是这样，我仍然觉得这个项目有很大的提升空间。受制于时间和设备原因，我短时间内无法做进一步的突破。留待有缘人来接手吧 ~（其实更大可能直接无了）~
